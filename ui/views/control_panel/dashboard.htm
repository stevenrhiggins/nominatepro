Control Panel

Here's a quick overview of what you can do in the Awards Control Panel.

  • Create a new award

    To create a new award:

    • Click the “Create a New Award” menu item at the top right of this page.
    • Enter the name of the award, such as “Employee of the Month”.

    You can also clone an award to save time when you have awards with similar information.

  • Important Dates (required)

    In this section, you can:

    • Set the start and end dates for nominations.
    • Set the start and end dates for judging (optional).
  • Award Details (required)

    In this section, you can:

    • Enter a welcome message.
    • Provide a description of the award.
    • Outline eligibility requirements.
    • Specify the award or prize for the successful nominee, if applicable.
    • Give general instructions for answering the questionnaire portion of the nomination.
    • Create a success message shown after a nomination is submitted.
    • Describe the criteria for judging (optional).
  • Nominator/Nominee Settings (required)

    This section offers several helpful options:

    • Allow users to self-nominate.
    • Notify nominees about their nomination, with or without revealing the nominator.
    • Require nominees to submit personal information, if needed.
  • Notification Settings

    This section allows you to compose emails to:

    • Alert nominators when their nomination is finished.
    • Notify nominees about their nomination or prompt them to submit personal details.

    These emails are sent based on the options you select in “Nominator/Nominee Settings”.

  • Contact Information

    Provide contact information to include in emails sent to nominators and nominees. You can also use this information as the sender for all outgoing emails.

  • Document Allowances & Instructions

    In this section, you can:

    • Set the maximum number of files nominators and nominees can upload (if nominee info is required).
    • Write file-upload instructions shown to users.
  • Sections (required)

    Create sections that organize questions for nominators. For example, a “Membership” section can group questions related to organizational membership.

  • Questions (required)

    In this area, you will:

    • Write questions for nominators and nominees (if nominee info is required).
    • Add an unlimited number of questions, including follow-up questions.

    If you use the Judging module, you can also complete a scoring rubric for each question to assist evaluation.

  • Nominator/Nominee Information

    This application always requires for the names and email addresses of nominators and nominees. You can also require more information:

    • Prefix and Suffix
    • Business/Association/Organization name
    • Position or Title
    • Address (Street, Unit/Apt, City, State/Province, Zip/Postal Code, Country)
    • Phone Number
    • Alternate Phone Number
  • Sponsor Logo

    Assign a sponsor’s logo to an award to integrate their branding into nominations. You can:

    • Upload a logo
    • Add a link to the sponsor’s website
    • Add an acknowledgement (e.g., “This nomination is sponsored by…”)
  • Restrict Access Option

    Limit who can participate in the nomination by requiring an access code. You can:

    • Use an access code generated by this application.
    • Provide your own instructions or use the provided template.
