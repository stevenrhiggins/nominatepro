Award Setup Checklist

  3. Complete all the categories in the General Settings section of the menu.
  5. General Settings:
    In this section, you can:
    • Set the start and end dates for nominations.
    • Enter a welcome message.
    • Provide a description of the award.
    • Outline eligibility requirements.
    • Specify the award or prize for the successful nominee, if applicable.
    • Create a success message that participants will see upon completing a nomination.
  6. Nominator/Nominee Settings:
    This section offers several helpful options:
    • Allow users to self-nominate.
    • Notify nominees about their nomination, with or without revealing the nominator's information.
    Require nominees to submit personal information, if needed.
  7. Email Notifications:
    This section allows you to compose emails to:
    • Alert nominators when their nomination is finished.
    • Notify nominees about their nomination or prompt them to submit personal details.
    These emails will be sent based on the settings you've chosen in the 'Nominators/Nominees Settings' section.
  8. Contact Information:
    In this section, you can provide basic contact information that will be included in emails sent to nominators and nominees as a point of contact for assistance. Additionally, you have the option to use this information as the sender for all emails sent to users.
  9. File Permission Settings:
    In this section, you can:
    • Set the maximum number of files that nominators and nominees can upload (if nominees are required to submit information).
    • Write file upload instructions that will be visible to users when they need to upload documentation.
  10. Sections:
    Use this area to create sections that organize questions for nominators. For example, you might create a section called 'Membership' to group questions related to membership in your organization or association.
  11. Questions:
    In this area, you will:
    • Write all the questions for nominators and nominees (if you've chosen to require nominee information).
    • Add an unlimited number of questions, including follow-up questions.
    If you opt to use our Judging module, you can also:
    • Complete a scoring rubric for each question to assist in evaluating nominations.
  12. Nominator/Nominee Information:
    This application will always require nominators (and nominees) to provide the names and email addresses of those being nominated and those who are nominating. However, you may require more information. This section allows you to select from the following options to be completed by nominators.
    • Prefix and Suffix
    • Business, Association, Organization etc. name
    • Position or Title
    • Address, including Stree, Unit/Apt, City, State/Province, Zip/Postal Code, Country
    • Phone Number
    • Alternate Phone Number
  13. Judging Settings:
    In this section, you can:
    • Set the start and end dates for judging.
    • Write the criteria necessary.
    • Name the judges you want to participate.
    • Write an email to use when sending invitations to judges.
    • Set whether to permit blind judging.
    • Set whether to permit vote changing.
    If you have recorded the names and email addresses of judges, you can send the invitations in bulk to them.
